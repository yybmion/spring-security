# 🔐 Spring Security 학습 정리

**Spring Security**의 기본 개념부터 다양한 인증/인가 방식까지 학습하고 정리한 내용입니다.

## ✨ 학습 내용

1. **기본 개념**
   - Spring Security 아키텍처
   - 인증과 인가
   - Security Filter Chain

2. **인증 구현**
   - 폼 로그인
   - JWT 토큰 인증
   - OAuth2 소셜 로그인

3. **권한 관리**
   - Role 기반 권한
   - 권한별 접근 제어

## 🛠 개발 환경
- Spring Boot 3.2.1
- Spring Security 6.2.1
- MySQL
- Gradle
